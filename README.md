# Pelis-Route

App de catálogo de películas construida con React y React Router.

## Funcionalidades

- Listado de películas populares
- Búsqueda de películas
- Detalle de cada película
- Navegación con React Router v6

## Tecnologías

- React 19
- React Router v6
- Vite
- CSS Modules
- TMDB API

## Instalación
```bash
npm install
npm run dev
```
